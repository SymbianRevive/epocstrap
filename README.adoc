an EPOCROOT that is fully usable from Linux.

== Usage

----
$ git clone --single-branch https://github.com/SymbianRevive/ERA.git era/
$ cd era/
$ bin/epocstrap -h
$ bin/epocstrap -r ~/epocroot.ext4
----

Then, follow the instructions display on screen.
